# WARScribe CLI

**Command-line tools for WARScribe notation.**

Register units, record actions, and generate transcripts from your terminal.

## Installation

```bash
uv pip install git+https://github.com/vindicta-platform/WARScribe-CLI.git
```

## Quick Example

```bash
warscribe register --unit "Captain" --id CPT-01
warscribe action "[MOVE: CPT-01 -> Zone-A]"
warscribe transcript
```
